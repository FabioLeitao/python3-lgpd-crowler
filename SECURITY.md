# Security Policy

## Supported Versions

Any Python3 version after 3.6 (tested on 3.13),... 

* uv 
* pip 
* Flask
* Flask-SQLAlchemy
* ReportLab
* Psycopg2 or MySQL Connector (depending on your database)
* SQLite3

#### Prereqs

1. sudo apt install libsqlcipher-dev libsqlclient-dev libsqlite3-dev sqlite3 sqlite3-tools mariadb-plugin-connect python3-mariadb-connector dbconfig-mysql
2. uv venv
3. uv self update
4. uv install pip
5. uv pip install setuptools
6. uv pip install flask
7. uv pip install flask_sqlalchemy
8. uv pip install reportlab
9. uv pip install psycopg2-binary
10. uv pip install mariadb
11. uv pip install mysql
12. uv pip install oracledb
