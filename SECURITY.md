# Security Policy

## Supported Versions

Any Python3 version after 3.6 (tested on 3.13),... 

* uv 
* pip 
* Flask
* Flask-SQLAlchemy
* ReportLab
* Psycopg2 or MySQL Connector (depending on your database)
* SQLite3

## Reporting a Vulnerability

You can submit a vunerability issue by creating in Issues tab.
